GLOBAL NEWS INTELLIGENCE BOT (FREE)

Este bot envía noticias globales relevantes a Telegram y explica el posible impacto en Chile.

TECNOLOGÍA:
- Python
- RSS de medios confiables
- IA gratuita de HuggingFace
- GitHub Actions (servidor gratis)

PASOS PARA USARLO:

1. Crear bot en Telegram
   Hablar con @BotFather
   /newbot

2. Obtener tu CHAT_ID
   https://api.telegram.org/botTU_TOKEN/getUpdates

3. Crear repositorio en GitHub

4. Subir todos los archivos de este proyecto

5. Ir a Settings > Secrets > Actions

Crear:

BOT_TOKEN = tu token de telegram
CHAT_ID = tu chat id
HF_TOKEN = token gratis de huggingface (https://huggingface.co/settings/tokens)

6. Activar GitHub Actions

El bot correrá cada 4 horas automáticamente.
